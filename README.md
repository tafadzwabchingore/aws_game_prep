# AWS Game Prep

A fun and interactive web-based quiz application to practice AWS certification questions and track your progress.

## Features

- **Interactive Quizzes**: Multiple choice questions on AWS topics including:
  - Compute (EC2, Lambda, ECS, EKS, etc.)
  - Storage (S3, EBS, EFS, etc.)
  - Networking (VPC, Route53, CloudFront, etc.)
  - Security (IAM, KMS, GuardDuty, etc.)
  - Databases (RDS, DynamoDB, Aurora, Redshift, etc.)
  - Architecture & Best Practices

- **Immediate Feedback**: Get explanations for correct and incorrect answers
- **Progress Tracking**: Your quiz history and performance metrics are saved locally
- **Badge System**: Earn badges based on your quiz performance:
  - 🥇 Gold: 90% or higher
  - 🥈 Silver: 75-89%
  - 🥉 Bronze: 50-74%

- **Statistics**: Track your daily study time and visit frequency
- **Flexible Selection**: Choose specific topics or mix all topics together
- **Customizable**: Select how many questions you want to practice with (1-60)

## How to Use

1. Open `aws_game.html` in your web browser
2. Select a topic or choose "All Topics (Mixed)"
3. Set the number of questions you want to practice
4. Click "Start Quiz" or "Random Quick Quiz"
5. Answer each question and get immediate feedback
6. Review your results and badges
7. All data is stored locally in your browser for privacy

## Features Explained

### Home Panel
- **Choose topic**: Select from individual AWS topics or mix all topics
- **Number of questions**: Choose between 1 and 60 questions
- **Start Quiz**: Begin a quiz with your selected parameters
- **Random Quick Quiz**: Start a 10-question quiz with random topics

### Progress Panel
- View your recent quiz results
- Check your daily time spent studying
- View your total days visited
- Clear all data when needed

### Quiz Area
- See your current progress (question X of Y)
- Track your score in real-time
- Get instant feedback on your answers
- Read explanations for correct answers

### Badges
- View your badge collection
- Check badge counts by level

## Data Privacy

All quiz data is stored locally in your browser using localStorage. No data is sent to external servers. You have full control over your data with the option to clear everything at any time.

## Browser Compatibility

This application works best in modern browsers that support:
- HTML5
- CSS3
- JavaScript ES6+
- localStorage

## Getting Started

Simply open the `aws_game.html` file in your web browser. No installation or server setup required!
